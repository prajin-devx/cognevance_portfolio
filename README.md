# 🚀 Cognevance Portfolio Website

A modern, responsive full-stack portfolio website developed as part of the **Cognevance Technologies Full Stack Web Developer Internship**.

The project showcases my skills, projects, and achievements while implementing frontend and backend integration through a contact form that stores user messages in a database.

---

## 📌 Project Overview

This portfolio website is designed to provide a professional online presence while demonstrating full-stack web development skills.

The application includes a responsive user interface, smooth navigation, project showcase, and a backend-powered contact system.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/cognevance_portfolio.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd ../server
npm install
npm run dev
```
